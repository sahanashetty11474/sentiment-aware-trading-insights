# Sentiment-Aware Trading Insights

This project analyzes how Bitcoin market sentiment (Fear vs Greed) influences trader behavior and performance on Hyperliquid.

## Objective
Understand whether sentiment regimes impact trader PnL distribution and trading behavior patterns.

## Methodology
- Cleaned and aligned sentiment and trader datasets
- Standardized timestamps at daily level
- Engineered features like win indicators and trade size segments
- Merged sentiment with trader activity
- Performed exploratory analysis across sentiment regimes

## Key Insights
- PnL dispersion varies across sentiment regimes
- Sentiment influences behavioral patterns in trading
- Larger trades show higher PnL variability

- Pandas
- Seaborn
